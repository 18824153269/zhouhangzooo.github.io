github提交没有贡献解决方案

清明时节雨无声，编程路上继续行。

前几天在github提交代码，发现没有显示贡献，于是查找原因顺便记录一下：原因可能是提交时的邮箱和github账号的邮箱不一致。

1.检查自己的emal是否正确

git config –global user.email 回车

2.设置邮箱名和名字

git config –global user.name “你的名字”
git config –global user.email “你的邮箱” 如XXXX@163.com
